КУРС ДОЛЛАРА - ВАРИАНТ 15
===============================================

ОПИСАНИЕ ПРОЕКТА:
-----------------
Приложение Windows Forms для учета курса доллара по месяцам в течение года.

ФУНКЦИОНАЛЬНОСТЬ:
-----------------
1. Ввод данных о курсе доллара за каждый месяц
2. Отображение данных в таблице
3. Сохранение данных в текстовый файл
4. Загрузка данных из текстового файла
5. Построение столбчатой диаграммы динамики курса

ТРЕБОВАНИЯ:
-----------
- .NET 6.0 или выше
- Windows OS

ЗАПУСК ПРОЕКТА:
--------------
1. Откройте проект в Visual Studio 2022 или выше
2. Нажмите F5 для запуска приложения

ИЛИ через командную строку:
1. Откройте терминал в папке проекта
2. Выполните команду: dotnet build
3. Выполните команду: dotnet run

ИСПОЛЬЗОВАНИЕ:
--------------
ВКЛАДКА "Ввод данных":
- Введите год (например: 2024)
- Выберите месяц из списка
- Введите курс доллара в рублях
- Нажмите "Добавить"

ВКЛАДКА "Таблица данных":
- Просмотр всех введенных данных
- Кнопка "Сохранить" - сохранение данных в файл
- Кнопка "Загрузить" - загрузка данных из файла

ВКЛАДКА "Диаграмма":
- Кнопка "Построить диаграмму" - отображение столбчатой диаграммы
- Диаграмма показывает динамику изменения курса по месяцам

ФОРМАТ ФАЙЛА ДАННЫХ:
-------------------
Строка 1: Год
Строка 2: Количество записей
Далее для каждой записи:
  - Название месяца
  - Курс доллара

Пример файла смотрите в example_data.txt

СТРУКТУРА ПРОЕКТА:
-----------------
- DollarCourse.csproj - файл проекта
- Program.cs - точка входа приложения
- Form1.cs - основная форма и логика
- Form1.Designer.cs - дизайн формы
- CurrencyData.cs - класс для хранения данных о курсе за месяц
- ArrayCurrencyData.cs - класс для работы с массивом данных
- example_data.txt - пример файла с данными
- README.txt - данный файл
