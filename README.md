## Описание работ

### Тема 1: Вычисление функции с помощью разложения в ряд

**Вариант 15:** Вычисление функции e^(-x²) через разложение в ряд Тейлора

**Формула:**
```
e^(-x²) = Σ(n=0 to ∞) [(-1)^n * x^(2n)] / n!
```

**Функциональность:**
- Вычисление суммы ряда с заданной точностью
- Использование рекуррентных формул для оптимизации
- Проверка корректности ввода данных
- Вывод результатов и сравнение с Math.Exp()
- Работа с отладчиком Visual Studio

**Файлы:**
- `Form1.cs` — основная логика программы
- `Алгоритм.md` — подробное описание алгоритма с блок-схемами
- `Содержание_отчета.md` — шаблон для создания отчета в DOCX

## Как использовать материалы

### Для создания проектов в Visual Studio:

1. **Создайте новый проект Windows Forms App (.NET Framework или .NET)**
2. **Скопируйте соответствующие .cs файлы в проект:**
   - Form1.cs
   - Form1.Designer.cs
   - Program.cs
   - Дополнительные файлы (для каждой темы свои)

3. **Для Темы 2 создайте отдельный проект Unit Test:**
   - Добавьте UnitTests.cs
   - Добавьте ссылку на основной проект

4. **Соберите и запустите проект**

### Для создания отчетов:

1. **Откройте файл `Содержание_отчета.md`** в соответствующей папке
2. **Создайте новый документ Word**
3. **Скопируйте структуру из MD файла**
4. **Добавьте скриншоты работы программы:**
   - Запущенная программа
   - Примеры вычислений
   - Результаты тестирования
   - Работа с отладчиком (для Темы 1)
   - Результаты юнит-тестов (для Темы 2)
   - Анимация движения (для Темы 3)

5. **Оформите согласно требованиям:**
   - Добавьте титульный лист
   - Добавьте верхний колонтитул с фамилией и группой
   - Пронумеруйте страницы
   - Создайте оглавление
