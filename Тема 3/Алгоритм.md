# Алгоритм движения квадрата по периметру формы

## Вариант 15

### Задача
Создать программу движения квадрата по периметру формы (не проходя нижнюю границу), в прямом и обратном направлении, с изменением цвета на каждом направлении движения. Окончание работы - нажатие любой клавиши.

---

## 1. Общее описание алгоритма

### Параметры квадрата:
- **squareX, squareY** — координаты левого верхнего угла квадрата
- **squareSize** — размер квадрата (40 пикселей)
- **squareColor** — текущий цвет квадрата
- **direction** — направление движения (0 = вправо, 1 = вниз, 2 = влево, 3 = вверх)
- **speed** — скорость движения в пикселях за такт
- **isMoving** — флаг активного движения

### Цвета для направлений:
- Направление 0 (вправо): Красный (Red)
- Направление 1 (вниз): Синий (Blue)
- Направление 2 (влево): Зеленый (Green)
- Направление 3 (вверх): Оранжевый (Orange)

---

## 2. Алгоритм движения квадрата

### Блок-схема основного цикла:

```
Начало такта таймера
   ↓
direction = ?
   ↓
┌────────────┬────────────┬────────────┬────────────┐
│ 0 (вправо) │ 1 (вниз)   │ 2 (влево)  │ 3 (вверх)  │
└────────────┴────────────┴────────────┴────────────┘
   ↓             ↓            ↓            ↓
squareX += speed  squareY += speed  squareX -= speed  squareY -= speed
   ↓             ↓            ↓            ↓
Достигли       Достигли     Достигли     Достигли
правой         нижней       левой        верхней
границы?       границы?     границы?     границы?
   ↓ (да)        ↓ (да)       ↓ (да)       ↓ (да)
direction = 1  direction = 2  direction = 3  direction = 0
ChangeColor()  ChangeColor()  ChangeColor()  ChangeColor()
   ↓             ↓            ↓            ↓
   └─────────────┴────────────┴────────────┘
                      ↓
              Перерисовка формы
                      ↓
                   Конец такта
```

## 3. Детальный алгоритм движения

### Псевдокод:

```
ФУНКЦИЯ MoveSquare():
    margin = 10
    bottomLimit = ClientHeight - margin

    ВЫБОР direction:
        СЛУЧАЙ 0:  // Движение вправо
            squareX = squareX + speed
            ЕСЛИ squareX + squareSize >= ClientWidth - margin ТО
                squareX = ClientWidth - margin - squareSize
                direction = 1
                ChangeColor()
            КОНЕЦ ЕСЛИ

        СЛУЧАЙ 1:  // Движение вниз (до предела)
            squareY = squareY + speed
            ЕСЛИ squareY + squareSize >= bottomLimit - squareSize ТО
                squareY = bottomLimit - squareSize - squareSize
                direction = 2
                ChangeColor()
            КОНЕЦ ЕСЛИ

        СЛУЧАЙ 2:  // Движение влево
            squareX = squareX - speed
            ЕСЛИ squareX <= margin ТО
                squareX = margin
                direction = 3
                ChangeColor()
            КОНЕЦ ЕСЛИ

        СЛУЧАЙ 3:  // Движение вверх
            squareY = squareY - speed
            ЕСЛИ squareY <= margin ТО
                squareY = margin
                direction = 0
                ChangeColor()
            КОНЕЦ ЕСЛИ
    КОНЕЦ ВЫБОРА
КОНЕЦ ФУНКЦИИ

ФУНКЦИЯ ChangeColor():
    colors = [Red, Blue, Green, Orange]
    squareColor = colors[direction % 4]
КОНЕЦ ФУНКЦИИ
```

## 4. Алгоритм отрисовки

```
ФУНКЦИЯ OnPaint(e):
    g = e.Graphics

    // Заливка квадрата текущим цветом
    brush = новая кисть(squareColor)
    g.FillRectangle(brush, squareX, squareY, squareSize, squareSize)

    // Рисование контура квадрата
    pen = новое перо(Black, толщина=2)
    g.DrawRectangle(pen, squareX, squareY, squareSize, squareSize)
КОНЕЦ ФУНКЦИИ
```

## 5. Обработка событий

### Запуск/Остановка движения:

```
ФУНКЦИЯ button1_Click():
    ЕСЛИ НЕ isMoving ТО
        timer1.Start()
        button1.Text = "Остановить"
        isMoving = true
    ИНАЧЕ
        timer1.Stop()
        button1.Text = "Запустить"
        isMoving = false
    КОНЕЦ ЕСЛИ
КОНЕЦ ФУНКЦИИ
```

### Нажатие клавиши (остановка):

```
ФУНКЦИЯ Form1_KeyPress(e):
    ЕСЛИ isMoving ТО
        timer1.Stop()
        isMoving = false
        button1.Text = "Запустить"
        Показать сообщение("Движение остановлено")
    КОНЕЦ ЕСЛИ
КОНЕЦ ФУНКЦИИ
```

### Изменение размера формы:

```
ФУНКЦИЯ Form1_Resize():
    margin = 10
    maxX = ClientWidth - margin - squareSize
    maxY = ClientHeight - margin - squareSize - squareSize

    ЕСЛИ squareX > maxX ТО squareX = maxX
    ЕСЛИ squareY > maxY ТО squareY = maxY

    Перерисовать форму
КОНЕЦ ФУНКЦИИ
```

## 7. Траектория движения

```
┌──────────────────────────────────────────────────────┐
│                                                       │
│  (margin, margin) ──────────► (maxX, margin)         │
│       ▲                             │                 │
│       │ Red                   Blue  │                 │
│       │                             ▼                 │
│       │                      (maxX, maxY)            │
│       │                             │                 │
│  Orange│                   Green    │                 │
│       │                             ▼                 │
│  (margin, margin) ◄────────── (margin, bottomLimit)  │
│                                                       │
│                                                       │
│ [Нижняя граница не используется]                     │
└──────────────────────────────────────────────────────┘

Направления:
0 → вправо  (Red)
1 ↓ вниз    (Blue)
2 ← влево   (Green)
3 ↑ вверх   (Orange)
```

## 8. Особенности реализации

### Двойная буферизация:
```csharp
this.DoubleBuffered = true;
```
Предотвращает мерцание при перерисовке формы.

### Обработка клавиатуры:
```csharp
this.KeyPreview = true;
```
Позволяет форме перехватывать нажатия клавиш.

### Адаптация к изменению размера:
При изменении размера формы квадрат не выходит за новые границы, координаты корректируются автоматически.

## 9. Таймер

- **Interval**: 50 мс по умолчанию (настраивается от 20 до 180 мс)
- **Tick**: вызывает MoveSquare() и Invalidate()
- **Управление**: запуск/остановка через button1 или нажатие клавиши

---

## 10. Последовательность выполнения

1. **Инициализация:**
   - Установка начальных координат квадрата
   - Установка начального цвета (Red)
   - Установка начального направления (0 - вправо)

2. **Запуск:**
   - Пользователь нажимает кнопку "Запустить"
   - Таймер начинает работу

3. **Движение:**
   - Каждый такт таймера:
     - Изменяются координаты квадрата
     - Проверяются границы
     - При достижении границы меняется направление и цвет
     - Форма перерисовывается

4. **Остановка:**
   - Пользователь нажимает "Остановить" или любую клавишу
   - Таймер останавливается
   - Квадрат остается на текущей позиции

5. **Настройка:**
   - Пользователь открывает форму настроек
   - Выбирает цвет и скорость
   - Применяет изменения

---

## 11. Граничные условия

### Правая граница:
```
squareX + squareSize >= ClientWidth - margin
```

### Нижняя граница (не доходя до самого низа):
```
squareY + squareSize >= bottomLimit - squareSize
```

### Левая граница:
```
squareX <= margin
```

### Верхняя граница:
```
squareY <= margin
```
