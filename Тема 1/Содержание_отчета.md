## 3. Математическая формулировка задачи

### 3.1 Описание входных данных
- **x** — аргумент функции (вещественное число, |x| < ∞)
- **ε** — допустимая погрешность вычислений (вещественное число, 0 < ε < 1)

### 3.2 Описание выходных данных
- Значение функции e^(-x²), вычисленное с помощью встроенной функции
- Значение функции, вычисленное через сумму ряда
- Количество просуммированных членов ряда
- Абсолютная погрешность вычислений

### 3.3 Функциональные характеристики программы
Программа должна:
1. Вычислять значение функции e^(-x²) через разложение в ряд:
   ```
   e^(-x²) = Σ(n=0 to ∞) [(-1)^n * x^(2n)] / n! = 1 - x² + x^4/2! - x^6/3! + ...
   ```
2. Использовать цикл с неизвестным числом повторений
3. Вычислять сумму ряда с заданной точностью ε
4. Проверять корректность ввода данных
5. Блокировать кнопку "Вычислить" при пустых полях ввода
6. Выводить результаты вычислений для сравнения

## 6. Текст программного модуля

Полный код программы находится в файлах:
- `Form1.cs` — основная логика программы
- `Form1.Designer.cs` — описание интерфейса
- `Program.cs` — точка входа приложения

### Основные методы:

**1. ExpFunction(double x)** — вычисление значения e^(-x²) с помощью встроенной функции Math.Exp

**2. Sum(double x, double eps, out int count)** — вычисление значения функции через разложение в ряд
   - Параметры:
     - x — аргумент функции
     - eps — точность вычислений
     - count — количество просуммированных членов ряда (выходной параметр)
   - Возвращает: сумму ряда

**3. button1_Click** — обработчик нажатия кнопки "Вычислить"
   - Получает значения из полей ввода
   - Проверяет диапазон eps
   - Вычисляет результаты
   - Выводит результаты в label3

**4. textBox_TextChanged** — обработчик изменения текста в полях ввода
   - Блокирует/разблокирует кнопку "Вычислить"

**5. textBox1_KeyPress** — проверка вводимых символов для поля x
   - Разрешает: цифры, запятую, точку, минус, Backspace
   - Заменяет точку на запятую

**6. textBox2_KeyPress** — проверка вводимых символов для поля eps
   - Разрешает: цифры, запятую, точку, Backspace
   - Заменяет точку на запятую
