# Алгоритм вычисления функции e^(-x²) через разложение в ряд

## Вариант 15

### Математическая модель

**Дано:**
- x – аргумент функции
- ε (eps) – допустимая погрешность, 0 < ε < 1

**Найти:**
- Значение функции e^(-x²)
- Сумму ряда, приближающую это значение
- Количество членов ряда, необходимых для достижения заданной точности

**Разложение в ряд:**

```
e^(-x²) = Σ(n=0 to ∞) [(-1)^n * x^(2n)] / n! = 1 - x² + x^4/2! - x^6/3! + x^8/4! - ...
```

### Рекуррентные формулы

Для оптимизации вычислений используются рекуррентные формулы:

1. **Начальные значения:**
   - sum = 1.0 (первый член ряда)
   - a = 1.0 (текущий член ряда)
   - count = 0 (счетчик членов ряда)
   - x² = x * x

2. **Рекуррентная формула для вычисления следующего члена ряда:**
   ```
   aₙ = aₙ₋₁ * (-x²) / n
   ```

3. **Накопление суммы:**
   ```
   sumₙ = sumₙ₋₁ + aₙ
   ```

4. **Условие выхода из цикла:**
   ```
   |aₙ| < ε * |sumₙ|
   ```

### Блок-схема алгоритма вычисления суммы ряда

```
Начало
   ↓
Ввод: x, ε
   ↓
sum := 1.0
a := 1.0
count := 0
x² := x * x
   ↓
┌─────────────────────┐
│ |a| ≥ ε * |sum| ?  │
└─────────────────────┘
   ↓ (да)          (нет) →
count := count + 1          ↓
   ↓                     Вывод: sum, count
a := a * (-x²) / count      ↓
   ↓                      Конец
sum := sum + a
   ↓
   └──→ (повторить цикл)
```

### Описание алгоритма

1. **Инициализация:**
   - Устанавливаем начальные значения: sum = 1, a = 1, count = 0
   - Вычисляем x² для оптимизации

2. **Цикл накопления суммы:**
   - Проверяем условие: |a| ≥ ε * |sum|
   - Если условие выполнено:
     - Увеличиваем счетчик: count = count + 1
     - Вычисляем следующий член ряда: a = a * (-x²) / count
     - Добавляем член к сумме: sum = sum + a
   - Иначе — выходим из цикла

3. **Результат:**
   - sum — приближенное значение функции
   - count — количество просуммированных членов ряда (без учета первого)

### Проверка корректности

Для проверки корректности вычислений программа сравнивает:
- Результат разложения в ряд (sum)
- Результат встроенной функции Math.Exp(-x²)
- Вычисляет абсолютную погрешность

### Область допустимых значений (ОДЗ)

- x: любое вещественное число (|x| < ∞)
- ε: 0 < ε < 1

### Пример расчета

**Входные данные:**
- x = 1
- ε = 0.0001

**Вычисления:**
1. sum = 1, a = 1, count = 0, x² = 1
2. Итерация 1: count = 1, a = -1, sum = 0
3. Итерация 2: count = 2, a = 0.5, sum = 0.5
4. Итерация 3: count = 3, a = -0.16667, sum = 0.33333
5. ... (продолжаем до достижения точности)

**Результат:**
- e^(-1) ≈ 0.3678794412
- Сумма ряда ≈ 0.3678794412 (с точностью ε)
